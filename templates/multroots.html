Multiple Roots


A multiple root corresponds to a point where a function is tangential to the \(x\) - axis. When the root is repeated as for example in \(x^2=0\), the root \(x=0\) is repeated. Graphically multiple roots can be seen when the curve touches tangentially to the \(x\) - axis. Multiple roots are repeated an even number of times when the function does not change sign, and an odd number of times when the function changes sign.
Its structure is very similar to that of Newton-Raphson, but the second derivative is required and can operate when the derivative is zero, even, making it more effective. Its equation is given as follows:
\[x_{n+1} = {x_{n}} - { f(x_n) \times f'(x_n) \over [f'(x_n)]^2 - f(x_n) \times f''(x_n)}.\]
